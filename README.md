# rec_listenradio

ListenRadio（リスラジ）の放送局を録音する、非公式のシェルスクリプトです。

> ※ 本スクリプト及び本ページは、ListenRadio とそのサービス及び株式会社ディーピーエヌとは一切関係がありません。

## 必要なもの

| コマンド | 用途 | 必須/任意 |
|---|---|---|
| `ffmpeg` | 録音・エンコード | **必須**（環境変数 `FFMPEG` で実行パスを上書き可） |
| `curl` | 局名・局リストの取得 | 任意（`-l` には必須） |
| `jq` | 上記 JSON の解析 | 任意（`-l` には必須） |

`curl` / `jq` が無くても録音自体は動作します（その場合、ファイル名・タグの局名は `ch<ID>` になります）。

## 使い方

```
$ ./rec_listenradio.sh [options]
```

### オプション

| オプション | 内容 |
|---|---|
| `-c <id>` | リスラジのチャンネルID（例: `30043`）※録音時は必須 |
| `-d <時間>` | 録音時間。数値だけなら「分」（例: `30`）、`:` を含めば `HH:MM:SS`（例: `01:30:00`）※録音時は必須 |
| `-f <mp3\|m4a>` | 出力形式（既定: `mp3`） |
| `-b <bitrate>` | 音声ビットレート kbps（既定: `128`）。`m4a` のときだけ `copy` を指定すると再エンコードせず元の AAC をそのまま保存（最高音質・`-b` の数値は無視） |
| `-o <file>` | 出力ファイル名（既定: `<局名>_<日時>.<拡張子>`） |
| `-l` | 局リスト（エリア別の「ID 局名」）を表示して終了 |
| `-n` | ドライラン（実行する ffmpeg コマンドを表示するだけ） |
| `-h` | ヘルプを表示 |

### 例

```sh
# 局リストを表示してチャンネルIDを調べる
$ ./rec_listenradio.sh -l

# mp3 / 128kbps で 30 分録音
$ ./rec_listenradio.sh -c 30043 -d 30

# mp3 / 256kbps、時間は HH:MM:SS、出力名を指定
$ ./rec_listenradio.sh -c 30043 -d 01:00:00 -f mp3 -b 256 -o radio.mp3

# m4a で AAC を無劣化コピー（最高音質）
$ ./rec_listenradio.sh -c 30043 -d 60 -f m4a -b copy
```

- 配信側の地域制限により、**日本国内からのみ**聴取・録音できます。
